1. 文件系统接口

1.1 列出文件或文件夹

  • 接口路径:/api/fs/list
  • 请求体 (JSON): 
    {
  "path": "字符串或数组", // 目标路径
  "type": 1, // 1: 扫描文件夹, 2: 扫描文件
  "exts": "jpg,png" // (可选) 仅在 type 为 2 时有效，筛选文件后缀，用逗号隔开
}
  • 返回状态码:
    • 200 OK:成功返回文件或文件夹列表数据。

1.2 获取文件内容

  • 接口路径:/api/fs/get
  • 请求体 (JSON): 
    {
  "path": "字符串或数组" // 文件相对路径
}
  • 返回状态码:
    • 200 OK:成功，返回文件二进制流 (Buffer)。
    • 404 Not Found:文件不存在。
    • 500 Internal Server Error:文件读取失败。

1.3 写入/上传文件

  • 接口路径:/api/fs/set
  • 请求头 (Headers):
    • File-Path: 目标保存目录
    • File-Name: 目标文件名（如果未提供，系统会自动生成 noname_时间戳.bin）
  • 请求体 (Body):
    • 文件的原始二进制数据 (Binary)。
  • 返回状态码:
    • 201 Created:文件写入成功。
    • 500 Internal Server Error:文件写入失败。

1.4 删除文件

  • 接口路径:/api/fs/del
  • 请求体 (JSON Array): 
    [
  "path/to/file1.txt",
  "path/to/file2.png"
]
  • 返回状态码:
    • 204 No Content:批量删除成功。
    • 404 Not Found:要删除的文件中存在不存在的文件。
    • 500 Internal Server Error:删除过程中出现底层错误。

2. 配置管理接口 (Settings)

权限注意:配置管理相关接口限制了访问源，仅允许 Host 为 127.0.0.1:{port} 或 localhost:{port} 的本地请求访问。

2.1 获取当前配置

  • 接口路径:/api/settings/get
  • 请求头 (Headers):
    • 需要携带正确的 Host。
  • 返回状态码:
    • 200 OK:返回全局配置对象（包含 sanjuuni, input, output, net 等参数）。
    • 403 Forbidden:非本地请求，拒绝访问。

2.2 修改/写入配置

  • 接口路径:/api/settings/set
  • 请求头 (Headers):
    • 需要携带正确的 Host。
  • 请求体 (JSON): 
    {
  // 欲修改的配置项键值对
}
  • 返回状态码:
    • 201 Created:配置修改成功。
    • 403 Forbidden:非本地请求，拒绝访问。
    • 500 Internal Server Error:配置参数校验失败或写入错误。

3. 任务/作业接口 (Job)

3.1 获取当前工作状态

  • 接口路径:/api/job/status
  • 返回体 (JSON): 
    {
  "inRunning": false, // 是否正在运行
  "inError": false, // 是否发生错误
  "time": 0, // 运行耗时
  "wait": [], // 等待队列
  "processing": [], // 正在处理队列
  "finish": [], // 已完成队列
  "ignore": [] // 忽略队列
}
  • 返回状态码:
    • 200 OK:成功返回状态。

3.2 开始执行任务

  • 接口路径:/api/job/start
  • 请求体 (JSON Array): 
    [
  // 需要处理的任务/图片数据列表
]
  • 返回状态码:
    • 201 Created:任务成功触发并转入后台异步运行（若传入空数组直接返回 201）。
    • 400 Bad Request:请求体不是一个有效的数组。
    • 503 Service Unavailable:当前已有任务在运行中，无法重复启动。

3.3 清除错误状态

  • 接口路径:/api/job/clearError
  • 说明:将作业状态中的 inError 重置为 false。
  • 返回状态码:
    • 201 Created:重置成功。
